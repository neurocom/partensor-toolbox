Detailed Description

template<typename Tensor_, typename ExecutionPolicy_ = execution::sequenced_policy, template< typename T > class DefaultValues_ = DefaultValues>
struct partensor::Status< Tensor_, ExecutionPolicy_, DefaultValues_ >

Returned Type of CPD algorithm.

Status is the returned type of cpd operations. In this struct exist the returned values, such as the cost function at the end of the algorithm, or at what iteration the operation has ended. Also, includes

the factors produced from cpd operation in an stl array of Matrix type and size same as the input Tensor order.

Template Parameters
Tensor_Type(data type and order) of input Tensor.
ExecutionPolicyThe policy that is used, either sequential or with mpi.
